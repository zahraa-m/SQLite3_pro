# What was the goal of this project?
1. Explore and prepare heart disease datasets of four different regions by using Pandas in python.
2. Design ER diagram to illustrate how datasets relate to each other in the RDBMS environment.
3. Produce a database of heart disease that follows three types of normalization (1NF, 2NF, and 3NF) by using SQLite3 in python.
4. Insert the datasets of four regions into the database.
5. Perform queries in the database.
